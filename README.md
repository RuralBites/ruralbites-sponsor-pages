# RuralBites Sponsor Pages

This repository contains customized follow-up pages for SFSP sponsors.

Each page is designed to:
- Show how RuralBites helps expand access to summer meals
- Reflect the sponsor’s local context
- Provide a clear and simple next step

## Structure

- templates/ → base templates
- sponsors/ → individual sponsor pages

## How to Create a New Page

1. Copy the template
2. Create a new folder under /sponsors/
3. Rename to the sponsor name (lowercase, hyphens)
4. Customize:
   - Sponsor name
   - Local context
   - Path forward
   - Next step
5. Publish and send link

## Notes

Do not include:
- routing logic
- internal systems
- operational workflows

Focus on:
- access
- clarity
- partnership
